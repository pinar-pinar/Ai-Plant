# PlantCare AI

This repository contains a local-first PlantCare AI proof-of-concept.

Structure:
- backend/: FastAPI backend with ML inference and training scripts
- frontend/: static frontend (HTML/JS/CSS) that talks to the backend

To get a ZIP of the repository, download from GitHub: https://github.com/pinar-pinar/Ai-Plant/archive/refs/heads/main.zip
