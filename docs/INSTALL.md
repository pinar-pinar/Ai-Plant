Installation et exécution locale

1) Clonez le repo
   git clone https://github.com/pinar-pinar/Ai-Plant.git
2) Lancez le script de démarrage (Linux/macOS)
   bash scripts/start_local.sh

Notes:
- Placez vos modèles dans backend/app/models/ (cnn_model.pt, symptom_model.pkl) pour des inférences réelles.
- Les scripts d'entraînement sont dans backend/app/ml/
